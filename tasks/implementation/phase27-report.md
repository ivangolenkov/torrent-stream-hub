# Отчет по фазе 27: Web GUI Responsive UX & Polish

## Что было сделано
1. **Адаптивность таблиц:**
   - В `TorrentTable.vue` переработана таблица торрентов. Теперь на мобильных экранах (`< md`) она скрывается, и вместо неё выводится список карточек торрентов (Responsive Cards), что исключает горизонтальный скролл.
   - В `BottomPanel.vue` (вкладка Files) применена аналогичная логика для списка файлов. На десктопе — удобная таблица, на мобилке — компактные карточки файлов с кнопками действий.

2. **Мобильный инспектор (Bottom Sheet):**
   - В `BottomPanel.vue` добавлено определение мобильного устройства (`window.innerWidth < 768`).
   - На мобильных устройствах отключен ресайз панели, а её высота зафиксирована на `85vh` (`isMobile ? {} : { height: \`\${panelHeight}px\` }`), что создает эффект выезжающего Bottom Sheet.
   - Добавлена анимация `slide-up` для плавного появления панели инспектора снизу.

3. **Адаптивное меню навигации:**
   - В `MainLayout.vue` текстовый логотип `TorrentStreamHub` скрывается на узких экранах.
   - Добавлена кнопка "Гамбургер" для мобильных экранов (`< md`), при клике на которую открывается выпадающее меню с навигацией (Dashboard, BT Health, About). Это предотвращает поломку шапки на малых разрешениях.

4. **Пустые состояния (Empty States):**
   - В `TorrentTable.vue` добавлено красивое "пустое состояние" (иконка `FolderOpenIcon` и кнопка `Add Torrent`), если массив торрентов пуст.
   - Кнопка "Add Torrent" теперь корректно "эмитит" событие, которое перехватывает родитель `Home.vue` для открытия модального окна.
   - В `BottomPanel.vue` (вкладка Files) облагорожен экран ожидания метаданных торрента с анимацией `ClockIcon`.
