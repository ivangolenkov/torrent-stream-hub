# Фаза 1: Инициализация проекта и базовая архитектура (Setup & Core)

На этом этапе закладывается фундамент приложения, настраивается структура директорий и конфигурация.

## Задачи

1. **Инициализация репозитория и модулей:**
   - Создать базовую структуру папок согласно разделу 9.1 ТЗ (Clean Architecture: `cmd`, `internal/config`, `models`, `engine`, `repository`, `delivery`).
   - Инициализировать Go модуль (`go mod init`).
   - Создать пустой проект Vue 3 + TypeScript + Vite в папке `web/`.
2. **Управление конфигурацией:**
   - Реализовать парсинг конфигурации в `internal/config` (переменные окружения ENV согласно разделу 10, значения по умолчанию, флаги запуска).
3. **Определение доменных моделей:**
   - Описать структуры данных в `internal/models/torrent.go`.
   - Задать константы для всех состояний конечного автомата (раздел 5.1: `Queued`, `Downloading`, `Streaming`, `Seeding` и т.д.).
   - Определить типы детализированных ошибок (раздел 5.2).

## Definition of Done (DoD)

- [ ] Создана базовая структура директорий бэкенда.
- [ ] Инициализирован `go.mod`.
- [ ] Инициализирован базовый Vue 3 проект в директории `web`.
- [ ] Написан код парсинга конфигурации (`internal/config`), поддерживающий все переменные из ТЗ (раздел 10).
- [ ] Определены структуры доменных моделей и константы состояний в `internal/models`.
- [ ] Код бэкенда компилируется без ошибок.
