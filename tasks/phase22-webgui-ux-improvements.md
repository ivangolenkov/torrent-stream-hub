# Phase 22: WebGUI UX/UI Improvements

## Описание
Улучшение пользовательского опыта (UX) и визуального оформления (UI) веб-интерфейса Torrent-Stream-Hub.

## Задачи
1.  **Индикаторы состояния (Badges):** Заменить текстовые статусы торрентов (`Paused`, `Downloading`, `Error`) на цветные бейджи для лучшей читаемости.
2.  **Карточная сетка статистики:** Переработать панель статистики (`DashboardStats`) в виде сетки карточек с улучшенной типографикой.
3.  **Реорганизация действий:** Сгруппировать кнопки управления торрентами (`Resume`, `Remove`) в выпадающее меню (dropdown) в таблице для разгрузки интерфейса.
4.  **Адаптивность:** Оптимизировать таблицу торрентов для мобильных устройств (скрытие/сворачивание колонок).
5.  **Тёмная тема:** Внедрить базовую поддержку тёмной темы через Tailwind.
6.  **Пустые состояния:** Добавить индикацию для случая, если список торрентов пуст.

*Примечание: Процент загрузки остается в текстовом виде (как сейчас).*

## Definition of Done (DoD)
- [ ] Все задачи по UX/UI внедрены и визуально проверены.
- [ ] Процент прогресса отображается текстом.
- [ ] Проект успешно собирается (`cd web && npm run build`).
- [ ] Написан отчет в `tasks/implementation/phase22-report.md`.
